🌍 Trip-Tales

Trip-Tales is a full-stack travel planning web application that allows users to create, manage, and organize their trips efficiently. The application provides authentication, trip management features, and a clean user interface for a smooth planning experience.

🚀 Features

-User Registration & Login Authentication

-Create, Edit, and Delete Trips

-Secure Session Management

-RESTful Routing Structure

-Responsive UI Design

-Organized MVC Project Structure
